# Sideshooter-Game-Python
## Description 
A game where you can shoot at enemies at the right of the screen in a scenario that is scrolling horizontally.

It's a simple example on how to use OOP principles.
### Topics:
- Classes 
- Basic Game Loop
- Sprites
- Key Inputs & Events
- Movements:
	- Controlled
	- Autonomous
- Collisions
- Game Mechanics
